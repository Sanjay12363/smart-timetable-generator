# Smart Timetable Generator (Backtracking)

## Project Overview
This project generates a timetable using backtracking by assigning subjects to time slots and labs without conflicts.

## Features
- Assign subjects to slots
- Avoid teacher conflicts
- Avoid lab clashes
- Generate valid timetable

## Technologies Used
- Java
- Backtracking Algorithm

## How to Run
1. Compile:
   javac Main.java
2. Run:
   java Main
